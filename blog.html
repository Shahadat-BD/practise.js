Discuss the scope of Var

  • var(variable) is a global scope . you can declared outside the any function
  • If you want to reassigned to variable value. you can do it
  • variable is hoisted.that means this hoisting will take variable moves to the top of the contain.if I don't set variable value. it's undefined(js seats default) .otherwise it will show variable value.
    example: 1. function hosted() { a = 4 var a; console.log(a);} hosted() // output : 4
    example: 2. function hosted() { var a; console.log(a);} hosted() // output : undefined
  • I can access anywhere of variable value.because it's a global scope
  • you can multiple time variable declared in same scoped

    • Discuss the scope of let

  • let are block scoped. thant means you can only access within the block . otherwise it wil be error throw
    example:1 function letter(){ let a = 4 ; console.log(a) } letter() // output = 4
    example:2 function letter(){ let a = 4 ; } console.log(a) letter() // output = a is not defined
  • you can be declared only one time within the same block scope.
  • let are not hosted in function. you can not access value before there declaratiion. otherwise it will show throw error.
    example : 1 function hoisted(params) { a = 4; let a console.log(a); } hoisted() /// output = error message
    example : 1 function hoisted(params) { let a a = 4; console.log(a); } hoisted() //output = 4
  • let can be reassigned but can not re declared . example 1 : let a = 4; a = 5 console.log(a) //output = 5;
    example 2 : let a = 4; let a = 5 console.log(a) //output = error;
